Tesla account

Sign in with your Tesla account to let the app download your invoices. The login happens on Tesla's own website; only the resulting token is stored (with strict file permissions) and used to fetch invoices. Your password is never seen by this app.

Status: -

Sync

New invoices for the current and previous month are fetched automatically. A full sync downloads the entire charging and subscription history and can take a while. If the automatic email export is enabled, you will be asked whether the invoices found by this sync should also be emailed (default: no).

Email backlog

Invoices synced while email export was disabled (or during a sync with email sending skipped) are marked as skipped and never emailed automatically. Send them here as a combined export — batched into a few emails instead of one mail per invoice.

Re-scan PDFs

Re-extracts cost and currency from the stored subscription PDFs with the current parser. Useful after an app update when older metadata contains wrong values. Charging figures come from the Tesla API and are refreshed by a sync instead.

Connect Tesla account
  1. Open the Tesla login in a new tab and sign in with your Tesla account. A security check may appear — that is Tesla's, not this app's.
    Open Tesla login
  2. After signing in, the browser shows an error or an empty page ("Page Not Found"). That is expected — the login worked; the page only fails because it tries to open the Tesla mobile app.
  3. Copy the full address of that page from the address bar — it starts with "tesla://auth/callback" — and paste it below:
Can't find the address? (common on desktop browsers) Some desktop browsers never show the tesla:// address in the address bar. Right-click the error page and choose "Inspect", open the "Console" tab, and look for a link starting with "tesla://auth/callback?code=" (it also appears on the "Network" tab). Copy that whole link and paste it below. If that is too fiddly, switch to "Paste refresh token" above.

If the browser sign-in does not work, you can create a refresh token with a trusted third-party tool — e.g. tesla_auth (Windows/macOS/Linux) or the "Auth app for Tesla" (iOS) — and paste it here. The token is verified with Tesla before it is stored.
